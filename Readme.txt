langkah-langkah untuk menjalankan aplikasi streamlit :

1. lakukan registrasi akun, dan lakukan login akun ke aplikasi untuk memasuki program utama
2. Untuk registrasi tidak dapat dilakukan dengan username yang sama, dan pada saat login, masukan username dan password yang sudah di registrasi, apabila   salah, maka login akan gagal.
3. saat berhasil registrasi dan berhasil login, anda akan diarahkan ke halaman utama, yaitu halaman aplikasi enkripsi dan dekripsi teks dan file
4. anda dapat memilih akan memakai program apa saja, ada 4 program yaitu XOR, RC4, DES,dan AES untuk program AES dan DES memiliki 3 mode yaitu, ECB,CBC dan CTR.
5. Setiap program mempunyai peryasratan "Key", dimana untuk mengenkripsi dan dekripsi pesan dan file, perlu key untuk melanjutkan programnya
6. Untuk XOR dan RC4 tidak ada ketentuan khusus untuk key nya, tetapi untuk AES dan DES memiliki ketentuan khusus
7. Anda dapat memilih akan mengenkripsi atau akan  mendekripsi teks atau file
8. Untuk file yang sudah di enkripsi dan di dekripsi dapat di unduh apabila prosesnya sudah selesai
9. hasil dapat anda lihat di library anda
